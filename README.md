# Concept in 60 Seconds

A full-stack AI-powered learning web app scaffold built with React, Tailwind, Express, and Firebase.

## Project structure

- `client/` — React frontend
- `server/` — Express backend

## Setup

1. Install dependencies for client and server:
   - `cd client && npm install`
   - `cd ../server && npm install`

2. Start development servers:
   - `cd client && npm run dev`
   - `cd server && npm start`

3. Open the frontend in your browser:
   - `http://localhost:5173`

## Notes

- The React frontend uses Vite and starts with `npm run dev`.
- The Express backend starts with `npm start` and listens on port `5000` by default.
- If port `5000` is already in use, stop the existing process or set `PORT` in `server/.env`.


This phase includes project scaffolding, routing, Tailwind configuration, and placeholder files for the AI, Firebase, and accessibility layers.
